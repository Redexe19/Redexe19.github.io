# Redexe19.github.io

Static GitHub Pages site for Redexe19's project notes, private showcases, and links.

## Structure

- `index.html` - main hub page
- `minecraft/index.html` - Minecraft project page
- `roblox/index.html` - Roblox project page
- `project.html` - shared project detail page
- `assets/css/style.css` - shared styling
- `assets/js/project-data.js` - shared project details
- `assets/js/*-data.js` - page-specific project groups

## Editing Cards

Most project cards are controlled by `assets/js/project-data.js`.
Cards link to `project.html?id=project-id`. Most entries are private, unreleased, or all rights reserved,
so the site does not include public release buttons.
